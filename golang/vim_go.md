# vim-go
golang vim plugin 

# vim-go 설치
    
    # vim-go bundle 다운로드
    $ cd ~/.vimrc/bundle
    $ git clone https://github.com/fatih/vim-go.git
    
    # .vimrc 설정
    $ vi ~/.vimrc
    Plugin 'fatih/vim-go'
    
    # 플러그인 설치
    $ vim 
    :PluginInstall
    

# vim-go Hello World!!
 - Run it
    ```
    :GoRun
    ```

 - Build it

     ```:GoBuild```
